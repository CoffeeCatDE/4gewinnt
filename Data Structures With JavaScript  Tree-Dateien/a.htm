                  /**
                  * Queue

                  function Queue() {
                  this.dataStore = []
                  this.enqueue = function enqueue(element) {
                  this.dataStore.push(element)
                  }
                  this.dequeue = function dequeue() {
                  return this.dataStore.shift()
                  }
                  this.front = function front() {
                  return this.dataStore[0]
                  }
                  this.back = function back() {
                  return this.dataStore[this.dataStore.length - 1]
                  }
                  }

                              Matt Cale 6 months ago

                              Fantastic article! Well communicated and built in the correct order (in my opinion). Thank you very much for being a teacher.

                              I think I have found a pretty minor mistake in you initial tree creation. If you look at the following snippet that borrows from your example you will see the inconsistency:

                              ```javascript
                              1 const tree = new Tree('1')
                              2
                              3 tree._root.children.push(new Node('2'))
                              4 tree._root.children[0].parent = tree
                              5
                              6 tree._root.children[0].children.push(new Node('5'))
                              7 tree._root.children[0].children[0].parent = tree._root.children[0]
                              ```

                              The parent for Node('2') [line 4] is the tree object, while the parent for Node('5') [line 7] is the Node('2') object. The way I see it a Node's parent ought to be another Node or null if you are dealing with the _root Node. The fix is simple, just replace the code on [line 4] with:

                              `tree._root.children[0].parent = tree._root`
